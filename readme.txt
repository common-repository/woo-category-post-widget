== Description ==

ONLY AVAILABLE FOR WooCoomerce

Woocommerce Category Post Widget WordPress plugin display it category wise products.
You can display products its category base.

= Descriptions of Widgets =

= Post Type (Product) =

display a list of the products by product category.

* **Title** - description that appears over the list of custom posts.
* **Number of products to show (at most unlimited)** - enter the number of products to display.
* **Display Product date?** - if checked, display product date.

== Installation ==
1. Download plugin file woocommerce-category-post-widget.zip, extract and upload folder woocommerce-category-post-widget to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress.
3. Adds widgets to a widget area and configure settings through the 'Widgets' menu in WordPress.
4. Have fun!
